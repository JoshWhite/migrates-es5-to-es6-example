# migrates-es5-to-es6-example

An example repository for low risk, gradual migration from ES5 namespaces to ES6 modules.

## How to use:

### Transpile your modular JS
- `cd src/modular`
- `npm i`
- `npm start`

### Test your modular JS

`npm test`

### Run the site

Open `./index.html` in your favorite browser
